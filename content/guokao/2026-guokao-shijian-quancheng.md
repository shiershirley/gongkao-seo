# 2026年国考时间安排：从报名到入职完整时间线

## 全年时间线总览

```
6-8月    → 岗位摸底、专业目录确认
10月中旬  → 公告发布 + 职位表公布
10月中下旬 → 网上报名（约7天）
11月底    → 笔试（公共科目）
12月-1月  → 成绩公布
2-3月     → 面试
3-4月     → 体检 + 政审
5-6月     → 公示 + 录用
7月       → 入职报到
```

## 各阶段详细说明

### 一、公告发布阶段（10月中旬）

- **发布公告**：国家公务员局官网（http://www.scs.gov.cn）
- **职位表下载**：中央机关及其直属机构考试录用公务员职位表
- **考试大纲**：公共科目笔试大纲 + 专业科目大纲
- **报考指南**：报名条件、流程、注意事项

> 关键动作：下载职位表，用Excel筛选符合条件的岗位，至少准备3-5个备选岗位。

### 二、报名阶段（10月中下旬，约7天）

| 时间 | 内容 |
|------|------|
| 第1-2天 | 提交报名申请 |
| 第3-4天 | 审核不通过可改报 |
| 第5-6天 | 缴费确认 |
| 第7天 | 打印准考证 |

**报名注意事项**：
- 不要卡最后一天报名（系统可能崩溃）
- 提前注册好账号，实名认证
- 照片需通过官方照片处理工具处理

### 三、笔试阶段（11月底）

#### 公共科目

| 科目 | 时间 | 题量 | 分值 |
|------|------|------|------|
| 行政职业能力测验 | 9:00-11:00 | 130-135题 | 满分100分 |
| 申论 | 14:00-17:00 | 4-5题 | 满分100分 |

#### 专业科目（部分岗位）

| 岗位类型 | 专业科目 | 时间 |
|---------|---------|------|
| 银保监/证监 | 经济金融/财会等 | 11月底下午 |
| 外交部等 | 外语水平测试 | 笔试前1-2天 |
| 公安类 | 公安专业科目 | 11月底下午 |

### 四、成绩公布（12月-1月）

- **笔试成绩**：行测+申论加权总分
- **最低合格分数线**：按地区和岗位类别划定
- **面试名单**：按1:3比例确定

### 五、面试阶段（2-3月）

| 事项 | 时间 | 说明 |
|------|------|------|
| 资格复审 | 面试前1-2周 | 提交证明材料 |
| 面试 | 2-3月 | 结构化/无领导 |
| 专业加试 | 部分岗位 | 面试当天或第二天 |

### 六、后续阶段

| 事项 | 时间 | 说明 |
|------|------|------|
| 体检 | 3-4月 | 按公务员体检标准 |
| 政审 | 4-5月 | 个人档案审核 |
| 公示 | 5月 | 5个工作日 |
| 录用 | 6月 | 发放录用通知 |
| 入职 | 7月 | 到岗报到 |

## 关键时间节点提醒

📅 **建议设闹钟的重要日期**：

1. 公告发布日 → 第一时间筛选岗位
2. 报名开始日 → 及时提交
3. 缴费截止日 → 切勿忘记
4. 准考证打印日 → 提前打印
5. 笔试日 → 提前踩点考场

## 备考时间建议

| 阶段 | 开始时间 | 重点关注 |
|------|---------|---------|
| 基础备考 | 6-7月 | 行测基础+申论入门 |
| 强化提升 | 8-9月 | 刷题+专项突破 |
| 冲刺模考 | 10月 | 模拟考试+查漏补缺 |
| 考前调整 | 11月 | 回顾错题+调整心态 |

> 2026年国考预计在2025年10月发布公告，如果你打算参加2026年国考，建议最迟在2025年6月开始备考。
