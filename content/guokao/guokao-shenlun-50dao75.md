# 国考申论怎么复习？从50分到75分的提升路线图

## 申论考试基本情况

| 项目 | 地市级 | 副省级 |
|------|--------|--------|
| 考试时间 | 180分钟 | 180分钟 |
| 题量 | 5题 | 5题 |
| 分值 | 100分 | 100分 |
| 大作文分值 | 40分 | 35分 |
| 难度 | ⭐⭐⭐ | ⭐⭐⭐⭐ |

## 申论得分分布

```
40分以下：基本没复习或完全偏题
40-50分：有基础但答题不规范
50-60分：大多数考生的水平
60-70分：认真备考的中上水平
70-80分：备考充分的高手
80分以上：极少数（前1%）
```

## 五大题型答题方法

### 一、单一概括题（10-15分）

**题型特征**："概括/总结/归纳……"

**答题公式**：
```
总括句（1句话）
  → 分项要点（①……②……③……）
  → 每个要点：关键词+简要展开
```

**评分标准**：
- 按点给分，每点2-3分
- 答出核心关键词即可得分
- 语言精炼，不要照抄原文

### 二、综合分析题（10-15分）

**题型特征**："谈谈你对……的理解/认识/看法"

**答题公式**：
```
表态/内涵（是什么）
  → 分析原因/影响/问题（为什么）
  → 总结对策（怎么办）
```

**常见类型**：
- 解释型分析：解释某句话的含义
- 评价型分析：对某现象/政策进行评价
- 比较型分析：比较A和B的异同

### 三、提出对策题（10-15分）

**题型特征**："针对……问题，提出解决措施/建议"

**答题公式**：
```
明确问题（1-2句）
  → 对策一：具体措施
  → 对策二：具体措施
  → 对策三：具体措施
  → 每条对策=手段+目的+预期效果
```

**注意**：对策要有针对性、可行性、可操作性

### 四、公文写作题（10-25分）

**常见公文类型**：
| 类型 | 常考文种 |
|------|---------|
| 宣传类 | 倡议书、公开信、宣传稿 |
| 总结类 | 工作汇报、总结报告、调研报告 |
| 方案类 | 活动方案、整改方案 |
| 建议类 | 建议书、指导意见 |

**答题公式**：
```
标题（居中）
  → 称呼（左上角）
  → 正文（开头+主体+结尾）
  → 落款（右下角）
  → 日期（右下角）
```

### 五、大作文（35-40分）

**评分标准**：

| 档次 | 分值 | 标准 |
|------|------|------|
| 一类文 | 30-40分 | 立意准确+论据充实+语言流畅+结构完整 |
| 二类文 | 20-30分 | 立意基本准确+有一定论据+结构完整 |
| 三类文 | 10-20分 | 立意偏题+论据不足+结构混乱 |
| 四类文 | 10分以下 | 跑题或未完成 |

**写作结构（五段论）**：
```
第一段（150字）：引出话题+亮明观点
第二段（250字）：分论点一+举例论证
第三段（250字）：分论点二+举例论证
第四段（250字）：分论点三+举例论证
第五段（150字）：总结升华+展望未来
```

## 从50分到75分的提升路线

### 第一阶段：50→60分（1个月）

**核心任务**：掌握答题规范

- 学习各题型答题格式
- 练习"找要点"能力（从材料中提取关键信息）
- 每周做1-2套完整申论
- 重点是**小题得分**（大作文先不求高分）

**关键突破**：
- 学会标注材料（圈画关键词、分段归纳）
- 答题必须分条列点（①②③④）
- 字迹工整，不涂改

### 第二阶段：60→70分（1个月）

**核心任务**：提升综合分析和大作文

- 练习综合分析的逻辑框架
- 积累10个以上通用论据素材
- 每周写1篇大作文并对照范文修改
- 学习范文的结构和表达

**关键突破**：
- 大作文学会从材料中提炼分论点
- 掌握"是什么-为什么-怎么办"的论述逻辑
- 积累社会治理、高质量发展等主题素材

### 第三阶段：70→75分（1个月）

**核心任务**：精细化打磨

- 限时模拟考试（180分钟完整做一套）
- 针对弱项专项突破
- 提升语言表达质量
- 大作文追求亮点（标题、开头、结尾）

**关键突破**：
- 申论语言从"大白话"到"规范用语"
- 学会运用排比、对仗等修辞
- 大作文标题和开头要有吸引力

## 申论备考常见误区

1. **误区一**：申论就是写作文 → 实际上小题占60%，大作文占40%
2. **误区二**：背模板就能高分 → 模板化作答最高只能得二类文
3. **误区三**：字数越多越好 → 答题要精炼，废话不加分
4. **误区四**：申论不用复习 → 不复习和认真复习差距可达20-30分
5. **误区五**：只练大作文不练小题 → 小题才是申论的"基本盘"

> 申论的秘诀是：**答案都在材料里**。学会从材料中提取、归纳、重组信息，比死记硬背模板有效得多。
