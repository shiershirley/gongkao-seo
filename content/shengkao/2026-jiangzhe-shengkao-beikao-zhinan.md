## 三地笔试科目

### 上海
- A类/B类：行测+申论+专业课（政法/综管/财经等）

### 江苏
- A类（综合管理）：行测+申论
- B类（技术类）：行测+申论
- C类（乡镇/街道）：行测+申论

### 浙江
- 行测+申论（参加联考）
- 部分岗位加试专业科目

## 各地报考策略

### 上海适合谁？

| 适合人群 | 原因 |
|---------|------|
| 在上海工作生活 | 避免异地调动 |
| 追求稳定体面 | 上海公务员社会地位高 |
| 有专业背景 | 政法/财经/信息等专业岗位多 |
| 应届名校生 | 上海对应届生较友好 |

### 江苏适合谁？

| 适合人群 | 原因 |
|---------|------|
| 追求高待遇 | 苏南公务员待遇全国第一梯队 |
| 江浙沪圈发展 | 交通便利、经济发达 |
| 不想去一线城市 | 苏南二线城市性价比高 |
| 本地人 | 各市岗位限制本地户籍较多 |

### 浙江适合谁？

| 适合人群 | 原因 |
|---------|------|
| 追求最高待遇 | 杭甬公务员待遇最好 |
| 互联网相关 | 数字化转型岗位多 |
| 浙江本地人 | 户籍限制较多 |
| 偏好联考 | 参加全国联考，资料丰富 |

## 最佳备考策略

### 策略：三地连考

```
时间线：
  12月：上海笔试 → 12月：江苏笔试
    → 3月：浙江联考笔试
```

三地可以同时备考，因为：
1. 笔试时间不冲突
2. 考试内容高度重合（行测+申论）
3. 备考一次、考三次

### 侧重点调整

| 时间 | 重点 |
|------|------|
| 10-12月 | 上海特色（科学推理+专业科目）|
| 12月 | 上海+江苏冲刺 |
| 1-3月 | 浙江省情+联考冲刺 |
