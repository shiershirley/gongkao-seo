## 真题示例解析

### 示例一：行测-言语理解

**题目**：社区居民普遍反映小区垃圾分类执行效果不佳，垃圾混投现象严重。作为社区工作者，你认为最有效的措施是：

A. 加大处罚力度，对违规居民罚款
B. 加强宣传教育，组织垃圾分类知识竞赛
C. 在每个楼道设置监控摄像头
D. 上报街道，等待上级指示

**答案：B**

**解析**：
- A：处罚过激，容易引发矛盾 ❌
- B：宣传教育是社区工作的首选方式，居民参与度高 ✅
- C：侵犯隐私 ❌
- D：推卸责任 ❌

### 示例二：社区专业知识

**题目**：根据《城市居民委员会组织法》，居民委员会是：

A. 基层政权组织
B. 基层群众性自治组织
C. 街道办事处的派出机构
D. 事业单位

**答案：B**

**解析**：《城市居民委员会组织法》第二条规定：居民委员会是居民自我管理、自我教育、自我服务的基层群众性自治组织。注意区分"自治组织"和"政权组织"。

### 示例三：案例分析题

**题目**：某老旧小区加装电梯，一楼居民强烈反对，三楼以上居民积极支持，双方多次发生激烈争吵。作为社区工作者，你如何处理？

**答题要点**：

1. **稳定情绪**：分别约谈双方代表，了解各自诉求
2. **摸清情况**：
   - 一楼反对原因（采光、噪音、补偿等）
   - 高层支持原因（出行便利、房产增值等）
3. **搭建平台**：召开居民听证会/协商会
4. **提出方案**：
   - 补偿方案（适当补偿低层居民）
   - 技术方案（选择低噪音电梯、优化采光设计）
   - 分摊方案（按楼层递增分摊费用）
5. **推动落实**：协助办理审批、签订协议
6. **跟踪回访**：施工期间做好协调，完工后回访

### 示例四：作文题

**题目**：以"社区工作者如何做好居民的贴心人"为主题，写一篇不少于500字的议论文。

**写作框架**：

```
标题：做居民的贴心人——新时代社区工作者的使命与担当

一、开头（100字）
  引入：社区是社会治理的最基层
  点题：社区工作者要做居民的贴心人

二、分论点一（150字）
  "贴心"体现在倾听居民心声
  举例：走访入户、民情日记

三、分论点二（150字）
  "贴心"体现在解决实际问题
  举例：加装电梯、垃圾分类、养老服务

四、分论点三（150字）
  "贴心"体现在创新服务方式
  举例：一网统管、数字化社区

五、结尾（100字）
  总结升华：社区虽小，但连接千家万户
  号召：以真诚之心服务每一位居民
```

## 备考建议

| 优先级 | 内容 | 投入时间 |
|--------|------|---------|
| ① | 社区法规文件 | 每天背诵30分钟 |
| ② | 行测资料分析 | 每天2篇 |

| ④ | 行测言语+判断 | 每天各15题 |
| ⑤ | 作文模板 | 每周1篇 |

1. 《城市居民委员会组织法》（全文）
2. 《上海市社区工作者管理办法》（全文）
3. 《关于加强和完善城乡社区治理的意见》（重点条文）
4. 《上海市社区治理条例》（重点条文）

## 考试策略

| 策略 | 说明 |
|------|------|
| 先易后难 | 先做行测单选，再做案例和作文 |
| 时间分配 | 严格按模块分配时间 |
| 不纠结 | 单选题不会的直接蒙，不要浪费时间 |
| 案例题分条作答 | ①②③④分条，条理清晰 |
| 作文字迹工整 | 卷面分很重要 |

> 上海社工笔试的难度适中，但社区专业知识是拉开分差的关键。把法规文件背熟、答题格式练好，加上行测基本功底，80分以上不是问题。
