# 行测类比推理真题解析：常见逻辑关系

**类比推理**是行测判断推理模块的常见题型，通常10道题左右。这类题目看似简单，但很容易选错。掌握常见的逻辑关系，是提高类比推理准确率的关键。

## 一、集合关系

### 全同关系

两个概念外延完全相同。

| 示例 | 关系 |
|------|------|
| 电脑：计算机 | 全同 |
| 母亲：妈妈 | 全同 |
| 土豆：马铃薯 | 全同 |

### 包含关系

一个概念外延完全包含另一个。

| 示例 | 关系 |
|------|------|
| 动物：猫 | 包含 |
| 水果：苹果 | 包含 |
| 电器：冰箱 | 包含 |

### 交叉关系

两个概念有重叠部分。

| 示例 | 关系 |
|------|------|
| 党员：教师 | 交叉 |
| 作家：诗人 | 交叉 |
| 医生：军人 | 交叉 |

### 并列关系

同一层级、同一类别的概念。

| 示例 | 关系 |
|------|------|
| 苹果：香蕉 | 并列 |
| 飞机：火车 | 并列 |
| 医生：教师 | 并列 |

## 五、真题演练

> **【2024年国考真题】** 窗户：玻璃

A. 衣服：扣子  B. 眼镜：镜片  C. 手机：屏幕  D. 电脑：键盘

**解析**：窗户包含玻璃，眼镜包含镜片。选B。

> **【2024年省考真题】** 生病：吃药

A. 饿了：吃饭  B. 考试：复习  C. 受伤：睡觉  D. 口渴：喝水

**解析**：生病需要吃药，饿了需要吃饭。因果顺承关系。选A。

> **【2023年省考真题】** 相机：拍照

A. 电视：观看  B. 钢笔：写字  C. 手机：通讯  D. 电脑：上网

**解析**：相机的主要功能是拍照，钢笔的主要功能是写字。选B。

## 六、秒杀口诀

1. **先横后纵**：先看横向关系（同一层级），再看纵向关系（层级高低）
2. **优先造句**：用选项造句，句式相同者更可能是答案
3. **注意层级**：注意是包含关系还是组成关系
4. **排除干扰**：有明显错误的选项直接排除

## 七、备考建议

1. **积累常识**：类比推理涉及各方面常识
2. **多做真题**：熟悉常见搭配
3. **注意细节**：细微差别往往决定答案
4. **控制时间**：每题不超过30秒

类比推理需要平时的积累和大量的练习！
