# 行测类比推理真题解析：常见逻辑关系

类比推理每道题字数不多，看起来简单，但很容易掉坑。这类题考的是你对概念之间关系的理解程度。把常见的几种关系搞清楚，做题就顺手多了。

## 一、集合关系

### 完全相同（全同关系）

两个词说的是同一个东西。

| 例子 | 关系 |
|------|------|
| 电脑：计算机 | 全同 |
| 母亲：妈妈 | 全同 |
| 土豆：马铃薯 | 全同 |

### 一个包含另一个（包含关系）

| 例子 | 关系 |
|------|------|
| 动物：猫 | 动物包含猫 |
| 水果：苹果 | 水果包含苹果 |
| 电器：冰箱 | 电器包含冰箱 |

### 你中有我我中有你（交叉关系）

| 例子 | 关系 |
|------|------|
| 党员：教师 | 有些党员是教师，有些教师是党员 |
| 作家：诗人 | 有些作家是诗人 |
| 医生：军人 | 有些医生是军人 |

### 平起平坐（并列关系）

| 例子 | 关系 |
|------|------|
| 苹果：香蕉 | 都是水果 |
| 飞机：火车 | 都是交通工具 |
| 医生：教师 | 都是职业 |

## 二、对应关系

### 功能对应

| 例子 | 关系 |
|------|------|
| 钥匙：锁 | 钥匙开锁 |
| 充电器：手机 | 充电器给手机充电 |
| 笔：写字 | 笔用来写字 |

### 材料对应

| 例子 | 关系 |
|------|------|
| 衣服：布料 | 布料做衣服 |
| 桌子：木头 | 木头做桌子 |
| 书：纸张 | 纸张做书 |

### 时间对应

| 例子 | 关系 |
|------|------|
| 春晚：除夕 | 春晚在除夕播出 |
| 月饼：中秋 | 月饼在中秋吃 |
| 粽子：端午 | 粽子在端午吃 |

### 因果对应

| 例子 | 关系 |
|------|------|
| 努力：成功 | 努力导致成功 |
| 骄兵：必败 | 骄兵必然失败 |
| 水涨：船高 | 水涨导致船高 |

## 三、语法关系

| 关系 | 例子 |
|------|------|
| 主谓 | 学生：学习、医生：看病 |
| 动宾 | 打扫：教室、修理：汽车 |
| 偏正 | 美丽：花朵、飞快：火箭 |

## 四、语义关系

| 关系 | 例子 |
|------|------|
| 近义 | 快乐：高兴、美丽：漂亮 |
| 反义 | 黑夜：白天、高：矮 |

## 五、真题演练

> **【2024年国考真题】** 窗户：玻璃

A. 衣服：扣子  B. 眼镜：镜片  C. 手机：屏幕  D. 电脑：键盘

**解析：** 窗户包含玻璃，眼镜包含镜片。B项关系一致。

---

> **【2024年省考真题】** 生病：吃药

A. 饿了：吃饭  B. 考试：复习  C. 受伤：睡觉  D. 口渴：喝水

**解析：** 生病需要吃药，饿了需要吃饭。因果顺承关系，选A。

## 六、解题技巧

1. **先横后纵**——先看横向关系是不是同类，再看纵向层级
2. **造句法**——把选项带进同一个句子，句式对得上的是答案
3. **注意层级**——是包含关系还是组成关系，别搞混
4. **排除法**——明显不对的直接排除

## 七、备考建议

1. 积累常见词语搭配
2. 多做真题，熟悉考试套路
3. 注意细微差别
4. 控制每题30秒以内

类比推理靠积累，平时多留意词语之间的各种关系！
