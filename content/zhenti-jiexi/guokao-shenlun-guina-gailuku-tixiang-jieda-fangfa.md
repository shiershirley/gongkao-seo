# 国考申论真题解析：归纳概括题型答题方法

**归纳概括题**是国考申论的基础题型，也是所有申论答题的根基。这类题目考查的是考生对材料的理解和提炼能力，看似简单，实则暗藏玄机。

## 一、题型特征识别

归纳概括题的标志性特征：

- 题干出现“概括”“归纳”“总结”“简述”等词
- 答案全部来源于给定材料
- 分值一般在10-20分之间

> **【2024年国考真题】** “给定资料2”中，L市的网格化管理在基层治理中发挥了重要作用。请简要概括其主要做法。（10分）

## 二、审题关键要素

### 1. 明确作答对象

- 概括什么问题？
- 是原因、问题、措施还是影响？
- 有没有限定范围（如“根据给定资料2”）？

### 2. 明确作答数量

- 是概括1个方面还是多个方面？
- 有没有字数限制？
- 要不要分类？

### 3. 明确层次要求

- 简单罗列还是分类归纳？
- 要不要加序号？

## 三、材料找点方法

### 核心技巧

1. **抓关键词**：问题词、原因词、影响词、对策词
2. **盯关联词**：因此、但是、然而、所以
3. **看高频词**：反复出现的表述往往是重点
4. **关注首尾句**：段落首尾往往包含核心信息

### 常见提示词

| 类型 | 提示词 |
|------|--------|
| 问题类 | 不足、缺陷、困境、难题 |
| 原因类 | 因为、由于、根源在于 |
| 对策类 | 应该、需要、必须、建议 |
| 意义类 | 有利于、促进了、实现了 |

## 四、答案组织技巧

### 结构要求

```
一、（总括词）+ 具体要点1
二、（总括词）+ 具体要点2
...
```

### 书写规范

- **踩点给分**：写关键词即可拿分
- **条理清晰**：用序号标注层次
- **控制字数**：超出字数会被扣分
- **不抄原文**：用规范语言概括

## 五、真题案例演示

> **【2024年国考真题】** 请概括F市在医养结合方面的主要经验。（15分）

**材料摘录**：
F市建立了以社区为平台、社会组织为载体、社工为支撑的“三社联动”机制，整合养老和医疗资源...

**答案示例**：
一、建立联动机制。以社区为平台、社会组织为载体、社工为支撑，整合养老和医疗资源。
二、强化政策支持。出台医养结合专项政策，加大财政投入。
三、培育专业队伍。加强医养结合专业人才培训，提升服务质量。

## 六、备考建议

1. **每日一练**：每天做一道归纳概括题
2. **复盘总结**：对照参考答案找出遗漏点
3. **积累规范词**：多读政府文件，学习规范表达
4. **控制字数**：练习时严格控制字数

归纳概括是申论的“基本功”，练好这个题型，其他题型都能受益！
