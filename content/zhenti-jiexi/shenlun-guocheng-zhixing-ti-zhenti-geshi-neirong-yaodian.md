# 申论贯彻执行题真题解析：格式与内容要点

贯彻执行题在申论考试里经常出现，要求你写一份符合规范的公文。这类题既考写作能力，又考格式规范，两者都不能马虎。

## 一、常考文种

| 文种 | 格式 | 什么时候用 |
|------|------|------------|
| 倡议书 | 标题+称谓+正文+落款 | 宣传动员 |
| 通知 | 标题+主送机关+正文+落款 | 布置工作 |
| 报告 | 标题+主送机关+正文+落款 | 汇报工作 |
| 方案/意见 | 标题+正文 | 提出建议 |
| 汇报材料 | 标题+正文 | 工作汇报 |

## 二、格式怎么写

### 完整格式（倡议书、通知等）

```
关于XXX的倡议书/通知/报告

各位居民/各社区/各部门：（顶格写）

正文内容……

落款（右对齐）
XX社区
2026年4月30日
```

### 简略格式（汇报材料等）

```
关于XXX的汇报材料

正文内容……
```

## 三、真题演示

> **【2024年国考真题】** 某社区准备开展“垃圾分类，从我做起”主题活动。请代社区居委会拟写一份倡议书。

**参考范文：**

```
关于开展“垃圾分类，从我做起”主题活动的倡议书

各位居民朋友：

为深入推进垃圾分类工作，共建美丽宜居社区，我们向全体居民发出如下倡议：

一、争做垃圾分类的践行者。从我做起，从家庭做起，按要求分类投放生活垃圾。

二、争做垃圾分类的宣传者。积极向家人、邻居、朋友宣传垃圾分类知识，营造良好氛围。

三、争做垃圾分类的监督者。主动监督身边人的分类行为，及时纠正错误分类。

让我们行动起来，用实际行动守护美好家园！

XX社区居委会
2026年4月30日
```

## 四、内容怎么写

### 开头部分

- 交代背景：为什么要写这个
- 说明目的：要达到什么效果
- 简洁有力，100字左右

### 主体部分

- 分条列项
- 措施要具体可操作
- 逻辑清晰

### 结尾部分

- 发出号召
- 简洁有力
- 100字以内

## 五、不同文种怎么写

### 倡议书

- 标题：关于XXX的倡议书
- 称谓：面向的对象
- 正文：背景+倡议内容+号召
- 落款：单位+日期

### 通知

- 标题：关于XXX的通知
- 主送机关：发文对象
- 正文：通知缘由+事项+执行要求
- 落款：单位+日期

### 汇报材料

- 标题：关于XXX的汇报
- 正文：工作开展情况+存在问题+下一步计划
- 不需要称谓和落款

## 六、容易出错的地方

| 错误 | 正确做法 |
|------|----------|
| 没有标题 | 必须写标题 |
| 标题不规范 | 用“关于...的...”格式 |
| 没有称谓 | 明确对象时要写 |
| 落款位置不对 | 右对齐，日期用阿拉伯数字 |

## 七、备考建议

1. 把各种文种的格式背熟
2. 积累常见的开头和结尾表述
3. 看清题目要求再写
4. 内容要紧扣材料

贯彻执行题格式和内容都很重要，缺一不可！
