# 申论贯彻执行题真题解析：格式与内容要点

**贯彻执行题**是国考申论的常见题型，通常要求考生根据给定材料，写一份符合规范的公文。这类题目不仅考查写作能力，还考查对公文格式的掌握。

## 一、常见文种

| 文种 | 格式要求 | 适用场景 |
|------|----------|----------|
| 倡议书 | 标题+称谓+正文+落款 | 宣传动员 |
| 通知 | 标题+主送机关+正文+落款 | 布置工作 |
| 报告 | 标题+主送机关+正文+落款 | 汇报工作 |
| 方案/意见 | 标题+正文 | 提出建议 |
| 汇报材料 | 标题+正文 | 工作汇报 |

## 二、格式规范

### 完整格式（标题+称谓+正文+落款）

适用于：倡议书、通知、报告、建议书等

```
【标题】
关于XXX的倡议书/通知/报告

【称谓】（顶格）
各位居民/各社区/各部门：

【正文】
一、（开头背景）
二、（主要做法/内容）
三、（号召/要求）

【落款】（右对齐）
XX社区
2026年X月X日
```

### 简略格式（标题+正文）

适用于：简报、汇报材料、短评等

```
【标题】
关于XXX的汇报材料

【正文】
...
```

## 三、真题案例

> **【2024年国考真题】** 某社区准备开展“垃圾分类，从我做起”主题活动。请你代社区居委会拟写一份倡议书。（20分）

### 参考答案

```
关于开展“垃圾分类，从我做起”主题活动的倡议书

各位居民朋友：

为深入推进垃圾分类工作，共建美丽宜居社区，我们向全体居民发出如下倡议：

一、争做垃圾分类的践行者。从我做起，从家庭做起，按要求分类投放生活垃圾。

二、争做垃圾分类的宣传者。积极向家人、邻居、朋友宣传垃圾分类知识，营造良好氛围。

三、争做垃圾分类的监督者。主动监督身边人的分类行为，及时纠正错误分类。

让我们行动起来，用实际行动守护美好家园！

XX社区居委会
2026年4月30日
```

## 四、内容要点

### 开头部分

- 交代背景（为什么要写这份公文）
- 说明目的（要达到什么效果）
- 简洁有力，150字以内

### 主体部分

- 分条列项，层次清晰
- 内容具体，可操作
- 措施有针对性

### 结尾部分

- 发出号召或提出要求
- 简洁有力，振奋人心
- 100字以内

## 五、常见文种写作要点

### 倡议书

- 标题：关于XXX的倡议书
- 称谓：面向的对象
- 正文：背景+倡议内容+号召
- 落款：单位+日期

### 通知

- 标题：关于XXX的通知
- 主送机关：发文对象
- 正文：通知缘由+通知事项+执行要求
- 落款：单位+日期

### 汇报材料

- 标题：关于XXX的汇报
- 正文：工作开展情况+存在问题+下一步计划
- 不需要称谓和落款

### 工作方案

- 标题：关于XXX的工作方案
- 正文：指导思想+工作目标+具体措施+保障机制
- 不需要称谓和落款

## 六、备考技巧

1. **熟记格式**：每种文种的格式必须背熟
2. **积累素材**：积累常见的开头和结尾表述
3. **审清题意**：看清题目要求的文种和字数
4. **结合材料**：内容要紧扣给定材料

## 七、易错提醒

| 错误类型 | 正确做法 |
|----------|----------|
| 缺少标题 | 必须有标题 |
| 标题不规范 | 使用“关于...的...”格式 |
| 称谓遗漏 | 明确对象时必须写称谓 |
| 落款位置 | 右对齐，日期用阿拉伯数字 |

贯彻执行题是“格式+内容”的综合考查，缺一不可！
