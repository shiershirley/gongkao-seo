## 考点一：增长率

### 必背公式

```
增长率 = 增量 ÷ 基期量 = （现期量 - 基期量）÷ 基期量

两期增长率比较：
  部分增长率 > 整体增长率 → 比重上升
  部分增长率 < 整体增长率 → 比重下降
```

### 真题示例

**题目**：2025年某市GDP为8500亿元，同比增长6.2%，其中第一产业增加值1200亿元，同比增长3.5%。2024年第一产业增加值占GDP的比重约为？

**解题思路**：

1. 判断比重变化：3.5% < 6.2%，比重下降 ✅
2. 计算现期比重：1200/8500 ≈ 14.1%
3. 估算基期比重（约上升0.3%）→ 约14.4%
4. 选最接近的选项

### 常见陷阱

| 陷阱类型 | 示例 |
|---------|------|
| 同比vs环比 | 题目问环比，材料给同比 |
| 增长率vs增长量 | 混淆"增长了百分之几"和"增长了多少" |
| 名义增长vs实际增长 | 名义GDP未扣除通胀 |

## 考点二：比重计算

### 必背公式

```
现期比重 = 部分 ÷ 整体

基期比重 = (部分÷(1+r1)) ÷ (整体÷(1+r2))
         = (部分/整体) × ((1+r2)/(1+r1))

比重变化 = (部分/整体) × ((r1-r2)/(1+r1))
```

### 真题示例

**题目**：2025年某省进出口总额5800亿元，同比增长8.5%，其中出口3500亿元，同比增长10.2%。2025年出口额占进出口总额的比重比上年？

**解题思路**：

1. 现期比重 = 3500/5800 ≈ 60.3%
2. r1（出口）= 10.2%，r2（整体）= 8.5%
3. 比重变化 = 60.3% × (10.2%-8.5%) / (1+10.2%)
4. ≈ 60.3% × 1.7% / 1.102 ≈ 0.93%
5. 比重上升约0.9个百分点

### 速算技巧：比重变化只需判断方向

| 条件 | 结论 | 需要精确计算？ |
|------|------|-------------|
| 只问"上升还是下降" | 比较 r1 和 r2 即可 | ❌ 不需要 |
| 问"变化了几个百分点" | 用公式算 | ✅ 需要 |

## 考点三：增长量计算

### 必背公式

```
增长量 = 现期量 - 基期量 = 现期量 × r/(1+r)
```

### n+1法速算

当增长率约等于 1/n 时：
```
增长量 ≈ 现期量 / (n+1)
```

| r ≈ | n | 增长量 ≈ |
|------|---|---------|
| 5% | 20 | 现期量/21 |
| 10% | 10 | 现期量/11 |
| 12.5% | 8 | 现期量/9 |
| 14.3% | 7 | 现期量/8 |
| 20% | 5 | 现期量/6 |
| 25% | 4 | 现期量/5 |
| 33.3% | 3 | 现期量/4 |

### 真题示例

**题目**：2025年某市社会消费品零售总额4200亿元，同比增长8.3%。比上年增加了多少亿元？

**解题思路**：

1. 8.3% ≈ 1/12
2. 增长量 ≈ 4200/(12+1) = 4200/13 ≈ 323亿元
3. 精确值：4200 × 8.3%/1.083 ≈ 322亿元
4. 速算足够接近，选322亿

## 考点四：综合判断题

### 解题策略

综合判断题通常是"以下说法正确/错误的是"，4个选项逐一判断。

**做题顺序**：
1. 先找**简单选项**（直接读数或简单计算）
2. 再找**中等选项**（需一步计算）
3. 最后处理**复杂选项**（需多步计算）

**时间控制**：综合判断题最多2分钟，不会就跳过。

### 常见错误选项特征

| 错误类型 | 特征 |
|---------|------|
| 偷换时间 | 题目问2025年，选项用2024年数据 |
| 偷换概念 | "进出口"vs"出口" |
| 单位陷阱 | "万元"vs"亿元" |
| 增长vs下降 | 把增长说成下降 |
| 比较对象错 | A和B比，选项说A和C比 |

## 高频速算方法总结

| 方法 | 适用场景 | 速度 |
|------|---------|------|
| 截位直除法 | 除法计算 | 最快 |
| 特殊分数法 | 增长率相关 | 快 |
| 尾数法 | 加减法 | 快 |
| 同位比较法 | 分数大小比较 | 快 |
| 差分法 | 分数大小比较 | 中等 |
| 错位加减法 | 复杂计算 | 慢但准 |
