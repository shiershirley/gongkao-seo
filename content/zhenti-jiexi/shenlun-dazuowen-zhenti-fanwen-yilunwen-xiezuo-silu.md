# 公考申论大作文真题范文：议论文写作思路

申论大作文是很多人头疼的部分，分值高，难度也不小。但其实大作文有固定的套路可循，只要掌握了框架，再配上合适的内容，拿个中上等分数并不难。

## 一、常用写作框架

### 五段三分法（最常见）

顾名思义，5段文字，3个分论点。

```
第一段：开头引入，提出论点
第二段：分论点1 + 论据
第三段：分论点2 + 论据
第四段：分论点3 + 论据
第五段：结尾总结，升华主题
```

### 开头怎么写

开头不要太长，150字左右就够了。

**常见写法：**
- 从材料里的现象说起
- 用一句有哲理的话引出主题
- 简述背景，点明论点

### 分论点怎么找

分论点一般在材料里会有提示，注意这几个地方：
- 关联词后面的内容（因此、所以、关键在于）
- 权威人士或文件里的话
- 反复出现的表述

### 结尾怎么收

结尾要简洁有力，100字左右。

**可以这样收：**
- 呼应开头
- 升华到更高层面
- 发出倡议或提出展望

## 二、真题示范

> **【2024年国考真题】** “基层治理，民生为本”。请写一篇议论文。

### 范文

**以民生为本，夯实基层治理之基**

基层治理是国家治理的基石，而民生是基层治理的根本所在。治理做得好不好，最终要看人民群众满不满意。从老旧小区改造到社区网格化管理，从“最多跑一次”到“一网通办”，这些改革措施的共同点就是以人民为中心。以民生为本，是新时代基层治理的根本遵循。

民生是基层治理的出发点。基层治理的直接对象是群众，治理效果好不好，群众最有发言权。上海某街道推行“社区晒账单”制度，把社区收支公开给居民看，主动接受监督，赢得了群众好评。基层治理只有把民生放在首位，才能获得群众的支持。

民生是基层治理的着力点。群众的操心事、烦心事，就是治理的发力点。杭州某社区针对老年人“数字鸿沟”问题，专门开设“手机课堂”，手把手教老人用智能手机。看似小事，实则关乎民生大计。

民生是基层治理的落脚点。治理成效如何，要体现在群众的笑脸上。成都某区推行“社区合伙人”制度，吸引社会组织、居民骨干、驻区单位共同参与，三年来居民满意度从72%提升到91%。让群众享受到治理红利，才能实现良性循环。

民惟邦本，本固邦宁。基层治理关系到国家长治久安，民生问题关系到千家万户幸福。让我们坚持以民生为本的治理理念，急民之所急、想民之所想、解民之所忧，夯实基层治理之基，绘就民生幸福画卷。

## 三、写作要点

| 要点 | 说明 |
|------|------|
| 标题 | 简洁明了，点明主题 |
| 开头 | 150字以内，引出论点 |
| 分论点 | 三个为宜，平行或递进 |
| 论据 | 结合材料+时政热点 |
| 结尾 | 升华主题，100字左右 |
| 字数 | 1000-1200字 |

大作文需要积累，平时多看时政评论、高质量范文，对写作很有帮助！
