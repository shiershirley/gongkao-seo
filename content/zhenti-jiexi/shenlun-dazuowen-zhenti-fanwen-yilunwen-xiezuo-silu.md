# 公考申论大作文真题范文：议论文写作思路

**申论大作文**是国考和省考的分值担当，通常占40-50分。很多考生对大作文望而生畏，其实只要掌握正确的写作框架，大作文并不难。本文详解议论文的写作思路，并提供高分范文。

## 一、经典写作框架

### 五段三分法（最常用）

```
第一段：引出论点（150字）
第二段：分论点1 + 论据（200字）
第三段：分论点2 + 论据（200字）
第四段：分论点3 + 论据（200字）
第五段：总结升华（100字）
```

### 总分总结构

```
总：开篇点题，明确论点
分：分层论述，逻辑清晰
总：收束全文，升华主题
```

## 二、真题题目

> **【2024年国考真题】** “给定资料”中说“基层治理，民生为本”。请深入思考这句话，联系实际，写一篇议论文。

**要求**：
- 观点明确，论据充实
- 逻辑清晰，语言流畅
- 不少于1000字
- 字迹工整，卷面整洁

## 三、高分范文

### 题目：以民生为本，夯实基层治理之基

### 范文正文

**第一段（引出论点）**

基层治理是国家治理的基石，民生是基层治理的根本所在。正如习近平总书记强调的“民生是最大的政治”，基层治理的一切工作都应围绕人民群众的切身利益展开。从社区网格化管理到老旧小区改造，从“最多跑一次”到“一网通办”，这些改革举措的共同特点就是以人民为中心、以民生为根本。**以民生为本**，是新时代基层治理的根本遵循。

**第二段（分论点1：民生是基层治理的出发点）**

民生是基层治理的出发点。基层治理的直接对象是人民群众，治理效果好不好，人民群众最有发言权。上海市某街道推行“社区晒账单”制度，将社区收支情况向居民公开，主动接受监督，赢得了群众的一致好评。这启示我们，基层治理只有始终把民生放在首位，才能获得群众的支持和认可。脱离群众的治理，无论形式多么华丽，都是无源之水、无本之木。

**第三段（分论点2：民生是基层治理的着力点）**

民生是基层治理的着力点。群众的操心事、烦心事、揪心事，就是基层治理的发力点。杭州市某社区针对老年人“数字鸿沟”问题，专门开设了“手机课堂”，手把手教老人使用智能手机，帮助他们跨越数字鸿沟。这一举措看似小事，实则关乎民生大计。基层治理就是要善于从小处着眼、从细节入手，用心用情解决好群众的“急难愁盼”问题。

**第四段（分论点3：民生是基层治理的落脚点）**

民生是基层治理的落脚点。治理成效如何，最终要体现在群众的笑脸上。成都市某区推行“社区合伙人”制度，吸引社会组织、居民骨干、驻区单位共同参与社区治理，形成了共建共治共享的格局。三年来，该区居民满意度从72%提升至91%。实践证明，基层治理的出发点和落脚点都是人民，只有让群众真正享受到治理红利，才能实现基层治理的良性循环。

**第五段（总结升华）**

“民惟邦本，本固邦宁。”基层治理关系到国家长治久安，民生问题关系到千家万户幸福。让我们坚持以民生为本的治理理念，急民之所急、想民之所想、解民之所忧，就一定能夯实基层治理之基，绘就民生幸福画卷。

## 四、写作技巧总结

| 要素 | 要求 |
|------|------|
| 标题 | 简洁明了，点明主题 |
| 开头 | 150字以内，引出论点 |
| 分论点 | 三个为宜，平行或递进 |
| 论据 | 结合材料+时政热点+自身积累 |
| 结尾 | 升华主题，100字左右 |
| 字数 | 控制在1000-1200字 |

大作文需要长期积累，多写多练才能得心应手！
