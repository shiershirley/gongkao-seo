# 2026省考行测申论高分备考全攻略：3个月冲刺计划与核心技巧

## 一、省考笔试科目与分值分布

### 行测（行政职业能力测验）

| 模块 | 题量 | 分值 | 建议用时 | 提分潜力 |
|------|------|------|---------|---------|
| 言语理解 | 35~40题 | 约35分 | 35分钟 | ★★★★ |
| 判断推理 | 30~35题 | 约30分 | 30分钟 | ★★★★★ |
| 资料分析 | 15~20题 | 约20分 | 25分钟 | ★★★★★ |
| 数量关系 | 10~15题 | 约10分 | 15分钟 | ★★★ |
| 常识判断 | 15~20题 | 约5分 | 10分钟 | ★★ |

### 申论

| 题型 | 题量 | 分值 | 建议 |
|------|------|------|------|
| 归纳概括 | 1~2题 | 10~15分 | 抓关键词，条理清晰 |
| 综合分析 | 1题 | 15~20分 | 多角度分析，逻辑严密 |
| 提出对策 | 0~1题 | 15~20分 | 对策可行，有针对性 |
| 贯彻执行 | 1题 | 20~25分 | 格式正确，内容完整 |
| 大作文 | 1题 | 35~40分 | 立意准确，论证有力 |

## 二、3个月备考时间规划

### 第1个月：基础阶段（系统学习）

**目标**：全面掌握各模块基础知识和解题方法

#### 行测部分

**第1~2周**：
- **言语理解**：学习主旨概括、细节判断、语句排序等方法
- **判断推理**：学习图形推理规律、逻辑判断方法、定义判断技巧

**第3~4周**：
- **资料分析**：学习速算技巧（截位直除法、特征数字法等）
- **数量关系**：学习高频题型（工程问题、行程问题、排列组合等）
- **常识判断**：每天积累30分钟，重点关注法律、政治、经济

#### 申论部分

- 学习申论基础题型和答题方法
- 每天阅读《半月谈》《人民日报》评论文章
- 每周完成1篇完整申论

### 第2个月：强化阶段（专项提升）

**目标**：各模块正确率达到目标水平

#### 行测目标正确率

| 模块 | 目标正确率 | 达标策略 |
|------|-----------|---------|
| 言语理解 | 80% | 大量刷题，总结常考词汇和出题规律 |
| 判断推理 | 85% | 熟练掌握各类推理方法 |
| 资料分析 | 90% | 提升计算速度，减少失误 |
| 数量关系 | 50%~60% | 放弃难题，确保简单题全对 |
| 常识判断 | 50% | 广泛积累，不过度投入 |

#### 申论目标

- 小题得分率达到70%以上
- 大作文稳定在28分以上（满分40分）

### 第3个月：冲刺阶段（真题+模拟）

**目标**：适应考试节奏，稳定发挥水平

- 每周完成2套完整真题或模拟题
- 严格按照考试时间进行全真模拟
- 分析错题，查漏补缺
- 申论每周完成2篇大作文

## 三、行测各模块核心技巧

### 言语理解与表达

**1. 主旨概括题**
- 找中心句（通常在段首或段尾）
- 关注意转折词："但是""然而""其实""不过"
- 排除绝对化表述的选项

**2. 逻辑填空题**
- 注意词语搭配习惯
- 辨析近义词的细微差别
- 结合语境推断空缺处含义

**3. 语句排序题**
- 找首句（引出话题的句子）
- 找捆绑关系（代词指代、关联词搭配）
- 验证尾句（总结性语句）

### 判断推理

**1. 图形推理**
- 优先看数量（点、线、面、角、素）
- 再看位置（平移、旋转、翻转）
- 最后看样式（叠加、求同求异）
- 记住常考规律：对称性、一笔画、曲直性

**2. 逻辑判断**
- **翻译推理**：熟练使用"前推后""后推前"
- **加强削弱**：找准论点，判断论据和论点的关系
- **日常推理**：关注必然性和可能性表述

### 资料分析

**速算技巧（必掌握）**：
1. **截位直除法**：分子分母同时截取前两位进行估算
2. **特征数字法**：将百分数转化为分数（如12.5%=1/8）
3. **尾数法**：适用于精确计算的加减运算
4. **同位比较法**：比较分数大小时只需比较前几位

**答题策略**：
- 先看问题和选项，再回材料找数据
- 注意时间、单位等"陷阱"
- 简单计算题确保全对

### 数量关系

**必做题型（性价比高）**：
- 工程问题：设总量为时间的最小公倍数
- 行程问题：画图辅助分析
- 容斥问题：利用公式或画韦恩图
- 排列组合：区分排列和组合，使用公式

**可放弃题型（性价比低）**：
- 复杂概率问题
- 高等数学相关
- 超出常规的"偏难怪"题

## 四、申论高分技巧

### 小题答题模板

**归纳概括题**：
```
总括句：概括核心要点
分述一：具体要点1（+材料关键词）
分述二：具体要点2（+材料关键词）
分述三：具体要点3（+材料关键词）
```

**综合分析题**：
```
表态/观点：对XX现象，我认为XX
原因分析：一是XX；二是XX；三是XX
影响分析：积极方面XX；消极方面XX
对策建议：因此，应该XX
```

**提出对策题**：
```
问题：XX方面存在XX问题
对策：一是完善制度，XX；二是加强监管，XX；三是XX
```

### 大作文写作技巧

**标题**：简洁有力，表明观点（如"以创新驱动基层治理现代化"）

**结构**：
- **开头段**（100~150字）：引出话题→分析问题→亮明观点
- **分论点一**（250~300字）：论点→论据→小结
- **分论点二**（250~300字）：论点→论据→小结
- **分论点三**（250~300字）：论点→论据→小结
- **结尾段**（100~150字）：总结全文→升华主题→展望未来

**素材运用**：
- 引用习近平总书记重要讲话精神
- 引用政策文件中的表述
- 引用具体案例和数据

## 五、考试当天实战策略

### 行测答题顺序建议

1. **常识判断**（10分钟）：快速作答，不会的直接猜
2. **言语理解**（35分钟）：逐题分析，确保准确率
3. **判断推理**（30分钟）：先做有把握的题型
4. **资料分析**（25分钟）：必须拿下，争取全对
5. **数量关系**（15分钟）：挑简单题做，难题跳过
6. **涂答题卡**（5分钟）：提前预留时间

### 申论答题时间分配

1. **阅读材料**（20分钟）：通读全部材料，标记关键信息
2. **小题**（80分钟）：每题15~20分钟，先做分值高的
3. **大作文**（60分钟）：构思10分钟，写作50分钟
