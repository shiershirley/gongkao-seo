# 公考行测速算技巧大全：资料分析必学10大方法

## 资料分析的重要性

资料分析是行测中**性价比最高**的模块：
- 题量稳定（20题/4篇）
- 技巧性强，掌握方法后提分最快
- 正确率目标：**85%以上**（至少做对17题）

## 10大速算技巧

### 技巧一：截位直除法（最常用）

**适用场景**：除法计算（求增长率、比重等）

**方法**：将分子分母各截取前几位进行估算

| 截位规则 | 选项差距 | 截取位数 |
|---------|---------|---------|
| 截2位 | 差距大（>10%） | 分子分母各取前2位 |
| 截3位 | 差距中（5%-10%） | 分子分母各取前3位 |
| 截4位 | 差距小（<5%） | 分子分母各取前4位 |

**示例**：
```
计算 34567 ÷ 89123 ≈ ?

选项：A.35%  B.38%  C.42%  D.45%

差距约5%-7%，截3位：
34567 ≈ 346，89123 ≈ 891
346 ÷ 891 ≈ 0.388 ≈ 38.8%

答案选B
```

### 技巧二：特殊分数法

**适用场景**：增长率/比重相关计算

**常用特殊分数对照表**：

| 分数 | 百分比 | 分数 | 百分比 |
|------|--------|------|--------|
| 1/2 | 50% | 1/6 | 16.7% |
| 1/3 | 33.3% | 1/7 | 14.3% |
| 1/4 | 25% | 1/8 | 12.5% |
| 1/5 | 20% | 1/9 | 11.1% |

**示例**：
```
某市GDP从3500亿增长到4700亿，求增长率？

增量：4700-3500=1200
1200/3500 = 12/35 ≈ 12/36 = 1/3 ≈ 33.3%

答案约34%
```

### 技巧三：尾数法

**适用场景**：加减法精确计算

**方法**：只看最后一位（或两位）判断答案

**示例**：
```
4567 + 7891 + 2345 = ?

尾数：7+1+5=13，尾数为3

答案尾数一定是3
```

### 技巧四：两期比重比较

**核心公式**：
```
部分增长率 > 整体增长率 → 比重上升
部分增长率 < 整体增长率 → 比重下降
部分增长率 = 整体增长率 → 比重不变
```

**只需比较两个增长率的大小，无需精确计算！**

### 技巧五：基期量速算

**公式**：基期量 = 现期量 ÷ (1+r)

**速算**：当r较小时（<5%），可近似为：
```
基期量 ≈ 现期量 × (1-r)
```

**示例**：
```
2025年产值为5800万元，增长率为3.2%，求2024年产值？

5800 ÷ 1.032 ≈ 5800 × 0.968 ≈ 5614万元
```

### 技巧六：年均增长速算

**公式**：年均增长率 ≈ 总增长率 ÷ 年数

**适用条件**：增长率较小时

**示例**：
```
2020年1000亿 → 2025年1500亿
总增长50%，5年
年均增长率 ≈ 50% ÷ 5 = 10%
（精确值为8.4%，估算够用）
```

### 技巧七：倍数与比重转换

**核心关系**：
```
A是B的n倍 → A占(A+B)的比重 = n/(n+1)
```

**示例**：
```
某省第一产业是第二产业的0.4倍
第一产业占比 = 0.4/(1+0.4) = 2/7 ≈ 28.6%
```

### 技巧八：增长量速算（n+1法）

**公式**：增长量 = 现期量 × r/(1+r)

**速算**：
- r > 0%：增长量 ≈ 现期量 × 1/(n+1)（其中1/n ≈ r）
- r < 0%：减少量 ≈ 现期量 × 1/(n-1)

**示例**：
```
3500亿增长14.3%（≈1/7）
增长量 ≈ 3500 × 1/(7+1) = 3500/8 = 437.5亿
```

### 技巧九：指数快速理解

| 指数值 | 含义 |
|--------|------|
| >100 | 比基期增长 |
| <100 | 比基期下降 |
| =100 | 与基期持平 |
| 105 | 比基期增长5% |
| 95 | 比基期下降5% |

### 技巧十：图形资料快速读数

**三步读图法**：
1. **看标题**：确定图表主题和时间
2. **看图例**：明确各图形代表的含义
3. **看单位**：注意单位换算（万/亿）
