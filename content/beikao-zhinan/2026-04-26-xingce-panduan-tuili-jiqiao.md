# 行测判断推理高分技巧：图形推理+逻辑判断+类比推理系统突破

## 判断推理板块概况

判断推理是行测中题量仅次于言语理解的第二大板块：

| 子题型 | 题量 | 难度 | 提分潜力 |
|-------|------|------|---------|
| 图形推理 | 10题 | 中等偏难 | ★★★★ |
| 定义判断 | 10-15题 | 中等 | ★★★★★ |
| 类比推理 | 10-15题 | 相对简单 | ★★★★★ |
| 逻辑判断 | 10-15题 | 较难 | ★★★ |

**战略优先级**：类比推理和定义判断最易提分，应优先保证得分率；逻辑判断相对复杂，适当分配精力。

## 第一部分：图形推理

图形推理考查对图形规律的识别能力，是很多考生的痛点。

### 核心解题路径

```
看图形类型 → 找变化规律 → 代入验证 → 选答案
```

### 五大核心规律

**规律一：元素个数变化**
- 观察每幅图中某类元素（线条、圆形、三角形等）的数量
- 常见规律：等差递增（1,2,3,4…）、等比（1,2,4,8…）

**规律二：元素位置变化**
- 元素在图中的位置（上下左右、内外）按规律移动
- 常见规律：顺时针旋转、对称翻转、平移

**规律三：叠加/相消**
- 多个图形叠加后，相同部分消失，不同部分保留
- 相同部分相加、不同部分相减

**规律四：对称性**
- 图形是否左右对称、上下对称、中心对称

**规律五：数量关系**
- 图中的点数、线段数、角的数量等存在规律
- 如：每行图形的面数之和相等

### 图形推理例题

**例**：观察以下图形规律，选出第四个位置的图形。

```
△ → ▲ → △▲ → ？
```

分析：白三角+黑三角，数量递增（1→1→2），下一个应为3个三角形组合。

## 第二部分：定义判断

定义判断给出一个概念的定义，要求判断哪个选项符合（或不符合）该定义。

### 核心方法：关键词提取法

**步骤**：
1. 快速阅读定义，**划出关键词**（主体、行为、结果、条件）
2. 对照每个选项，逐项核对关键词
3. 所有关键词都满足 → 符合定义

**重点**：定义判断不考你是否认同该定义，只考你能否准确理解并应用它。

### 定义判断典型例题

**例**：
> **社会惰化**：指个人在群体中工作时，由于责任分散而降低工作努力程度的现象。

以下哪个选项最符合"社会惰化"？

A. 小王独自完成任务时全力以赴，但在团队项目中总是依赖他人，贡献度明显降低
B. 大李在公开演讲时非常紧张，表现不如私下练习时好
C. 小张知道自己是项目唯一负责人，因此格外努力
D. 某公司部门裁员后，剩余员工工作积极性明显提高

**分析**：
- 定义关键词：①在群体中；②责任分散；③降低努力程度
- A：在团队中（群体）→ 依赖他人（责任分散）→ 贡献降低（努力程度降低）✓完全符合
- B：原因是紧张，不是责任分散 ✗
- C：恰好相反，是努力提高 ✗
- D：是因为人减少工作量增加，不符合 ✗

**答案**：A

## 第三部分：类比推理

类比推理考查词语之间的逻辑关系，要求找出与给定词对关系相同的选项。

### 常见关系类型

| 关系类型 | 示例 |
|---------|------|
| 包含关系 | 动物：老虎（老虎是动物） |
| 成分关系 | 盐：氯化钠（盐的化学成分是氯化钠） |
| 功能关系 | 眼镜：近视（眼镜用于矫正近视） |
| 因果关系 | 努力：成功（努力导致成功） |
| 场所关系 | 医院：护士（护士工作在医院） |
| 并列关系 | 玫瑰：百合（都是花） |
| 对反关系 | 高：矮（反义词） |

### 解题关键：精确描述关系

很多考生在类比推理中失分，原因是**关系描述不精确**，导致多个选项都"感觉对"。

**正确做法**：
1. 用完整句子描述词对关系（"A是B的…"/"A用于…"）
2. 找到最精确的关系描述
3. 用同样的句子套入选项，看哪个完全吻合

**例**：
> 教师：学校 → （？）：（？）

初步描述：教师在学校工作（场所关系）

选项：
A. 演员：舞台 ✓（演员在舞台表演，场所关系，最精确）
B. 书：图书馆 ✗（书存放在图书馆，不是工作关系）
C. 医生：患者 ✗（服务关系，不是场所关系）
D. 军人：武器 ✗（使用关系，不是场所关系）

**答案**：A

### 类比推理注意事项

1. **优先看形式相同**：如词对"A：B"，先找选项中形式相同的关系
2. **关系要完全对应**：不能只是"有点像"，要完全相同
3. **注意词序**：A→B和B→A可能关系不同
4. **排除法**：先排除明显不同的选项

## 第四部分：逻辑判断

逻辑判断（加强/削弱论证类）是判断推理最难的题型，但掌握方法后可以稳定得分。

### 三段论结构

大多数逻辑题可以分解为：
- **前提P1**：某个已知条件
- **前提P2**：另一个已知条件
- **结论C**：由前提推出的结论

**题目要求**：找到最能加强/削弱/支持结论的选项。

### 削弱型题目技巧

削弱结论的方式：
1. **否定前提**：前提不成立，结论就站不住脚
2. **切断联系**：说明前提和结论之间没有因果联系
3. **另有他因**：说明结论的出现有其他原因，与前提无关
4. **举出反例**：找到一个与结论相反的具体案例

### 加强型题目技巧

加强结论的方式：
1. **补充前提**：提供缺失的逻辑链条
2. **排除他因**：说明排除了其他可能原因，强化了因果联系
3. **数据支撑**：提供支持结论的额外证据

### 逻辑判断例题

**例**：
> 研究发现，经常吃早餐的学生，学习成绩普遍高于不吃早餐的学生。因此，研究者得出结论：吃早餐能提高学生的学习成绩。

以下哪项最能**削弱**上述结论？

A. 不吃早餐会导致低血糖，影响注意力
B. 实验中部分学生成绩优秀但不吃早餐
C. 吃早餐的学生通常有更好的家庭条件和更好的学习习惯，这些因素本身就有助于提高成绩
D. 学习成绩还受到智力、努力程度等多种因素影响

**分析**：
- 结论：吃早餐→成绩高（因果关系）
- 最好的削弱方式：说明存在**另一个原因**（有钱家庭+好习惯）可以同时解释"吃早餐"和"成绩高"，切断了因果关系
- C：指出混淆变量（好家庭→吃早餐；好家庭→好成绩），这是经典的"切断因果联系"

**答案**：C
