# 公考备考时间规划：在职备考6个月上岸全方案

## 在职备考的挑战

| 挑战 | 解决方案 |
|------|---------|
| 时间碎片化 | 利用早起+晚间+周末完整时间 |
| 精力有限 | 抓重点模块，不平均用力 |
| 信息不对称 | 关注权威公众号/官网 |
| 心理压力大 | 设定阶段性目标，小步前进 |

**每日时间投入**：约3.5小时
**周末时间投入**：约10小时
**周总投入**：约28小时

## 各模块时间分配策略

### 投入产出比排序

| 优先级 | 模块 | 原因 | 建议投入 |
|--------|------|------|---------|
| ① | 资料分析 | 提分快、回报高 | 每天必做 |
| ② | 言语理解 | 题量大、分值高 | 每天必做 |
| ③ | 判断推理 | 题量大、易突破 | 每天必做 |
| ④ | 申论小题 | 申论基本盘 | 每周3次 |
| ⑤ | 数量关系 | 投入大回报低 | 选做 |
| ⑥ | 常识判断 | 范围广难预测 | 碎片时间 |
| ⑦ | 申论大作文 | 需要长期积累 | 每周1篇 |

## 碎片时间利用指南

| 碎片时间 | 时长 | 建议做 |
|---------|------|--------|
| 通勤路上 | 30-60分钟 | 听申论素材音频/常识刷题 |
| 午休时间 | 30分钟 | 行测刷题（言语/判断） |
| 排队等候 | 5-10分钟 | 记忆公式/成语 |
| 睡前 | 15分钟 | 回顾当天错题 |

## 高效学习方法

### 方法一：错题本（最重要！）

建立电子错题本，记录：
- 题目截图
- 错误答案
- 正确答案
- 错误原因
- 解题技巧

每周回顾一次错题本，反复出现的错误重点攻克。

### 方法二：番茄工作法

- 25分钟专注学习 + 5分钟休息
- 每个番茄钟只做一件事
- 晚上3小时 = 约6个番茄钟

### 方法三：限时训练

- 所有练习必须计时
- 逐步缩短用时
- 行测单模块限时：资料分析25分钟、言语35分钟

## 在职备考心态管理

1. **接受不完美**：不可能每题都会做，学会取舍
2. **设定小目标**：每周设定可达成的目标
3. **定期放松**：每周末休息半天
4. **寻找战友**：加入备考群互相监督
5. **记录进步**：看到自己的提升，保持信心
