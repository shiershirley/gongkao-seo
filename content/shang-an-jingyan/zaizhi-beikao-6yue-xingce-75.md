# 在职备考6个月，行测75分上岸市直机关经验分享

## 个人背景

| 项目 | 情况 |
|------|------|
| 学历 | 普通二本，管理学专业 |
| 工作 | 互联网公司运营，朝九晚六 |
| 备考时间 | 每天3小时（早起1h+晚上2h），周末10小时 |
| 备考周期 | 6个月 |
| 考试目标 | 某省直机关（竞争比约80:1） |
| 最终成绩 | 行测75分、申论68分、面试82分，总分第二名上岸 |

## 备考时间安排

### 日常时间表

| 时间 | 周一至周五 |
|------|----------|
| 6:00-7:00 | 资料分析2篇+常识刷题30道 |
| 12:30-13:00 | 言语理解15道+判断推理10道 |
| 20:00-22:00 | 专项学习（1个模块）或申论练习 |
| 22:00-22:30 | 回顾当天错题 |

### 周末安排

| 时间 | 内容 |
|------|------|
| 周六上午 | 行测系统学习/模块刷题 |
| 周六下午 | 申论练习 |
| 周日上午 | 全真模拟考试 |
| 周日下午 | 模考复盘+错题整理 |

## 各模块提分历程

### 第1-2月：基础阶段

| 模块 | 初始正确率 | 2月后正确率 | 提分方法 |
|------|----------|----------|---------|
| 言语理解 | 55% | 70% | 每天刷30题+积累高频词 |
| 判断推理 | 50% | 65% | 学完基础理论后大量刷题 |
| 资料分析 | 40% | 65% | 学速算技巧+每天练2篇 |
| 数量关系 | 20% | 35% | 放弃难题，只做简单题 |
| 常识 | 35% | 40% | 碎片时间看时政 |

### 第3-4月：强化阶段

| 模块 | 2月正确率 | 4月后正确率 | 提分方法 |
|------|----------|----------|---------|
| 言语理解 | 70% | 78% | 总结错题规律，提高速度 |
| 判断推理 | 65% | 80% | 图推每天20题+逻辑每天10题 |
| 资料分析 | 65% | 85% | 限时训练，25分钟做完4篇 |
| 数量关系 | 35% | 50% | 掌握工程/利润/容斥3类 |
| 常识 | 40% | 45% | 重点看法律+省情 |

### 第5-6月：冲刺阶段

| 模块 | 4月正确率 | 最终正确率 | 考试策略 |
|------|----------|----------|---------|
| 言语理解 | 78% | 80% | 35分钟做完 |
| 判断推理 | 80% | 82% | 35分钟做完 |
| 资料分析 | 85% | 90% | 25分钟做完 |
| 数量关系 | 50% | 45% | 15分钟做8题，其余蒙 |
| 常识 | 45% | 50% | 5分钟快速过 |

## 行测75分的关键

### 关键一：资料分析必须拿高分

- 资料90%正确率 = 行测基本盘稳固
- 每天雷打不动练2篇资料分析
- 速算技巧必须熟练（截位直除法用得最多）

### 关键二：学会战略性放弃

| 放弃的 | 保留的 |
|--------|--------|
| 数量关系难题（每题超2分钟） | 数量关系简单题（工程/利润） |
| 常识不会的（5秒内没思路） | 有把握的常识题 |
| 言语纠结的（超过1分钟） | 有明显线索的言语题 |

### 关键三：严格控制时间

我的考试时间分配：
```
常识：5分钟（快做，不会就蒙）
言语：35分钟
判断：35分钟
资料：25分钟
数量：15分钟（做8题+蒙7题）
涂卡：5分钟
```

## 申论68分的经验

说实话我的申论不算高，但有以下心得：

1. **小题才是基本盘**：我归纳概括和公文写得比较规范，得分还可以
2. **大作文别追求完美**：结构清晰比文采更重要
3. **字迹要工整**：我练了1个月的字帖，卷面分有提升
4. **积累申论素材**：每周读3篇人民日报评论

## 面试82分的经验

面试翻盘是上岸的关键（笔试第三，面试第二）：

1. **开口练习最重要**：每天对着镜子练2道题
2. **不要背模板**：考官很反感千篇一律的回答
3. **举例接地气**：用自己生活中的例子更有说服力
4. **报了个面试班**：5天封闭训练，帮助很大

## 给在职考生的建议

1. **坚持早起**：早上的1小时效率最高
2. **利用通勤**：地铁上刷题（粉笔APP）
3. **周末全真模拟**：必须限时做全套
4. **找个搭子**：互相监督不容易放弃
5. **接受波动**：模考分数上下浮动是正常的
6. **相信自己**：在职备考确实辛苦，但坚持下来的都上岸了

> 我不是最聪明的，也不是学得最快的，但我可能是最坚持的。6个月、每天3小时，700个小时的投入，换来了一份稳定体面的工作。值得。
