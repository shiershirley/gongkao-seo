# 2026-04-29 工作日志

## 公考SEO Vercel Build失败修复

### 问题描述
4/28、4/29连续两天Vercel部署失败，本地build正常但线上报错YAMLException。

### 根本原因
4篇文章的frontmatter中description字段内嵌了未转义的英文双引号`"`，例如：
```yaml
description: "考生对报名流程还停留在"网上填表"这个粗浅认知上"
```
双引号嵌套导致YAML解析器报错。

### 修复方案
1. 定位4个问题文件
2. 将内嵌双引号替换为日文直角引号「」
